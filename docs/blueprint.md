# **App Name**: LeadTracker 2025

## Core Features:

- Dashboard: Presents consolidated sales metrics with comparison charts for enrollment vs. registration, conversion rates, and total students.
- Individual Candidate Lookup: Allows searching for a specific candidate by name or registration code, displaying corresponding data such as course and objectives.
- General Data Filtering: Provides a dynamic filtering page for the sales database, allowing users to filter based on candidate status, registration date, and enrollment date.
- Contact Export: Generates a list of selected contacts for export, including name, phone number, course, and registration ID.
- Weekly Agenda: Offers a weekly activity planner that dynamically generates days of the week based on a start date, listing weekly objectives and pending tasks.
- Lead Scoring Insights: Generates leads scoring by evaluating candidates’ attributes (courses or other fields of interest). Then evaluates against all of the past enrollment data, using this tool to find patterns of past success and prioritize efforts towards most promising leads. Adjustments to factors such as contact follow up type, offer pricing strategies.
- Import Data: Functionality to import data from a CSV file into the application.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to convey trust and professionalism, inspired by the consolidated nature of the dashboard metrics.
- Background color: Light gray (#F5F5F5), a desaturated shade of the primary hue for a clean, non-intrusive backdrop.
- Accent color: Electric Green (#76FF03) to highlight important actions and metrics, creating a sense of forward motion.
- Headline font: 'Space Grotesk' sans-serif for a computerized feel
- Body font: 'Inter' sans-serif for a modern, neutral look in the tables and forms.
- Use flat, modern icons to represent different data types and actions, ensuring clarity and usability.
- Design a responsive layout that adapts to different screen sizes, ensuring optimal viewing experience across devices.
